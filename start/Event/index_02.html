Patrones

Cocina

¿Has considerado alguna vez que al cocinar están ejecutando una serie de patrones? Que al cocinar un chef sigue una receta, que también se le puede llamar patrón de cocina. Las recetas son las instrucciones de un patrón que se pueden extender a otros campos.

Veliz

¿Alguna vez has considerado que empacar una maleta es un problema de optimización? Para empacar un veliz debemos decidir entre peso, tamaño y volumen. Incluso la ropa que seleccionamos requiere consideras todos los cambios que podamos formar.

Monedas

¿Alguna vez has considerado que el encontrar una moneda constituye un patrón de búsqueda? Para encontrar una moneda primero debes identificar el tipo de moneda, el tamaño de la moneda, el color de la moneda. Cada pregunta que estableciendo un patrón distinto de búsqueda.
